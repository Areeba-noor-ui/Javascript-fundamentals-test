## 🔍 Student Scores Program (Task 2)

This task focuses on practicing arrays, loops, functions, and string methods in JavaScript.  
It demonstrates how to process student data, calculate results, and implement basic searching logic.

---

## 📌 Concepts Covered (Task 2)
- Arrays and array indexing
- `for` loops
- Functions with parameters and return values
- Total and average score calculation
- Pass / Fail checking using conditions
- String methods:
  - `toLowerCase()` for case-insensitive comparison
- Modular and reusable code structure

---

## 🧠 What Task 2 Does
- Stores student names and their scores in arrays
- Displays each student’s score using loops
- Calculates:
  - Total score of all students
  - Average score
- Checks whether each student passed or failed
- Uses functions to organize logic clearly
- Applies `toLowerCase()` to make searching reliable regardless of letter case

---

## 💡 Why `toLowerCase()` Is Used
JavaScript is case-sensitive.  
To prevent search errors caused by capitalization differences, both user input and stored names are converted to lowercase before comparison.

This ensures accurate and professional searching behavior.
