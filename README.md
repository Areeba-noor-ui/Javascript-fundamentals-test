# JavaScript Fundamentals – Day 1 to Day 3
This repository contains my  JavaScript practice task where I implemented core JavaScript fundamentals and basic logic building.
The goal of this task is to understand how JavaScript works with variables, data types, operators, conditions, and basic decision-making.

## 📌 Concepts Covered
- JavaScript variables (`let`, `const`)
- Data types and `typeof` operator
- Template literals
- Logical operators (`&&`)
- Conditional statements (`if / else`)
- Ternary operator
- Truthy and falsy values
- Clean and readable console output
---
## 🧠 What This Program Does
- Stores user information such as name, age, education years, and agreement
- Checks and displays data types of each variable
- Applies multiple conditions to determine eligibility for a course
- Uses both `if-else` and ternary operator for decision making
- Prints a final eligibility result in the console
